## Improve homepage prompt

```text
Improve index.html and styles.css while keeping the existing static HTML/CSS/JS stack. Make the homepage feel more like a dark, warm, chaotic olive wood workshop. Do not add frameworks. Keep it mobile-first and readable.
```

## Improve gallery prompt

```text

```
