# Implementation Rules

## Allowed

- Semantic HTML
- CSS variables
- Vanilla JavaScript
- JSON data
- Static images
- Simple product filtering
- Simple product detail modal/page
- WhatsApp/email links

## Avoid

- Frameworks
- Build tools
- Payment integration
- Database
- Login system
- Complex animations
- Over-polished ecommerce UI

## Design rules

- Use dark workshop feeling as base
- Use warm wood tones
- Keep the layout readable despite the chaotic brand vibe
- Mobile first
- Product inquiry over checkout
- Images carry personality
- Text should be short and direct

## Accessibility basics

- Real alt text
- Buttons as buttons
- Links as links
- Enough color contrast
- Large tap targets on mobile
