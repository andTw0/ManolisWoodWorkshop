# 4. Image preparation rules

## Naming

Use clean names.

Bad:

```text
IMG_4928-final-final-copy.png
```

Good:

```text
hero-storefront.jpg
product-bowl-01.jpg
```

## Optimization

Prepare multiple sizes if possible:

```text
hero images: 1600–2200px wide
product cards: 800–1200px wide
mobile-friendly compressed JPG/WebP
```

## Important

The assignment requires fast mobile performance. Do not use massive uncompressed Facebook images directly in the prototype.
